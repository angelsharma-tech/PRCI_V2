# PRCI v2.0 - Visual UI Display

## Purpose

This Streamlit-based UI provides a visual demonstration interface for PRCI v2.0 academic procrastination support system. This is designed for academic presentation and demonstration purposes only.

## Features

- Clean, professional interface for academic demonstration
- Input field for academic situation description
- Visual output display with detection results
- Risk level visualization
- Intervention suggestion display
- Feedback collection interface
- Non-clinical disclaimer and safety information

## Installation and Running

### Step 1: Install Streamlit
```bash
pip install streamlit
```

### Step 2: Run the Application
```bash
cd F:/sem8/PRCI_v2/ui_display
streamlit run app.py
```

The application will automatically open in your default web browser at `http://localhost:8501`

## Usage

1. Enter your academic situation in the text area
2. Click "Analyze" to see demonstration results
3. Review the detection summary and risk level
4. Read the suggested academic support
5. Provide feedback on the helpfulness of the suggestion
